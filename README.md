# Dungeon Crawling
Jogo de nome provisório, cuja iniciativa foi tomada a partir da ideia de uma game jam com um amigo próximo. 

A ideia do jogo é ser um top-down, rogue like com níveis baseados em andares, altamente inspirado em Enter the Gungeon, Soul Knight e Binding of Isaac.

O prazo de conclusão é indefinido, mas está previsto para o fim de fevereiro.
